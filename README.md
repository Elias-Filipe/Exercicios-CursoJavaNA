# ☕ Exercícios em Java / Java Exercises

Este repositório contém **exercícios práticos em Java**, desenvolvidos com o objetivo de aprimorar a lógica de programação, praticar a linguagem e consolidar conceitos fundamentais da computação.

Os exercícios abordam diferentes temas, como:
- Estruturas de controle (`if`, `for`, `while`, etc.)
- Arrays e matrizes  
- Funções e métodos  
- Programação orientada a objetos (POO)  
- Estruturas de dados básicas (listas, filas, pilhas, árvores, etc.)  
- Entrada e saída de dados  
- Manipulação de arquivos  

> ⚙️ **Status:** Em desenvolvimento. Este repositório será atualizado conforme novos exercícios forem concluídos.

This repository contains **practical Java exercises**, developed with the goal of improving programming logic, practicing the language, and consolidating fundamental computer science concepts.

The exercises cover different topics, such as:
- Control structures (`if`, `for`, `while`, etc.)
- Arrays and matrices  
- Functions and methods  
- Object-oriented programming (OOP)  
- Basic data structures (lists, queues, stacks, trees, etc.)  
- Input and output handling  
- File manipulation  

> ⚙️ **Status:** In development. This repository will be updated as new exercises are completed.
